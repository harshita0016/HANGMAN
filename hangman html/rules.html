🎮 Welcome to Hang-The-Man

📜 Game Rules

🔹 Objective

The goal of Hang-The-Man is to correctly guess a hidden word before the man is hanged. Players must think carefully and use hints wisely to solve the word within the allowed attempts and time.

🔹 Basic Gameplay Rules

  • You start with 5 wrong attempts.
  • You must enter one letter at a time to guess the word.
  • If the guessed letter exists in the word, it will automatically appear in the correct positions.
  • If the guess is incorrect, one attempt is deducted and the hangman drawing progresses.
  • If all attempts are used, the player loses the game.
  • If the word is guessed before attempts run out, the player wins.

⏱ Time Limit

  • Each game round has a 120 second timer.
  • If the timer reaches zero before the word is guessed, the round automatically ends.

🎯 Difficulty Levels

The game provides three difficulty modes:

  • Easy – Simple and commonly used words.
  • Medium – Moderately challenging words.
  • Hard – Complex and longer words for experienced players.

🪙 Coin System

  • Players earn 10 coins for each correctly solved word.
  • Coins represent the player's score.
  • Coins can be used to purchase clues when needed.

💡 Clue System

  • Players begin with a few free clues.
  • Each clue reveals one hidden letter in the word.
  • If no clues remain, players can buy one using 5 coins.
  • Use clues strategically as they reduce your available coins.

👥 Multiplayer Mode

  • Players can join multiplayer rooms.
  • Up to 4 players can compete in a single game.
  • If the room fills up, the game starts immediately.
  • If the waiting timer expires and at least 2 players are present, the match begins.
  • The winner is determined by:
    • Fastest completion time
    • Fewest wrong attempts

😈 Game Personality

Unlike traditional hangman games, Hang-The-Man includes humorous and sarcastic commentary that appears as the player progresses through the game. These lines react to correct guesses, mistakes, and near failures to make the experience more entertaining.
