👻 About Hang-The-Man

🎯 A Smart Twist on the Classic Hangman Game

🔹 Game Overview

Hang-The-Man is a modern implementation of the traditional Hangman game developed as a Python-based GUI application. The game challenges players to guess hidden words using hints while managing limited attempts.

This game integrates multiple technologies including Tkinter, MySQL, Flask, and HTML to create an interactive gaming experience with both Singleplayer and Multiplayer modes.

🎮 Game Modes

  • Singleplayer Mode – Players guess words individually and earn coins based on correct answers.
  • Multiplayer Mode – Players join game rooms and compete to solve the same word fastest.

⭐ Key Features

  • Interactive GUI interface built using Tkinter.
  • Dynamic word and hint selection from a MySQL database.
  • Player login and signup system with secure password hashing (bcrypt).
  • Real-time leaderboard powered by Flask and HTML.
  • Multiplayer room system with waiting lobby.
  • Sound effects and animated hangman stages for better gameplay experience.
  • Three difficulty levels – Easy, Medium, and Hard – allowing players to choose the level of challenge based on their skill.
  • Humorous and sarcastic commentary lines that appear during gameplay depending on the player's progress, making the game more entertaining and engaging.
  • Coin and clue system where players earn coins for correct answers and can spend them to reveal letters.
