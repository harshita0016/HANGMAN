👻 About Hang-The-Man

🎯 A Fun Twist on the Classic Hangman Game 🎯

🔹 About the Game

Hang-The-Man is a word guessing game designed to be fun, challenging, and educational at the same time. The game randomly picks a word and you must guess it before your chances run out!

🔹 Features

  • Interactive GUI built with Tkinter.
  • Words and hints stored in a MySQL database.
  • Dynamic leaderboard powered by Flask and HTML.
  • Engaging visuals and humorous taglines while playing.
